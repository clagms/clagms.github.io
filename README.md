# Overview

This repository holds the sources to https://clagms.github.io/

## Website Maintenance

1. Install ruby and all dependencies for jekyl websites: https://jekyllrb.com/docs/
2. Update bundler
   ```
   bundle update --bundler
   ```
3. Update packages
   ```
   bundle update
   ```
4. Test serve
   ```
   bundle exec jekyll serve
   ```
